# HackDFW Buildathon
Welcome our github.

## Usage
Make a new branch for any hacking.
`git checkout -b "branch"`

Add new files
`git add "file"`

Commit new changes
`git commit -m "description of changes"`

Push changes
`git push -u origin master`

Bring in new changes
`git fetch #branch`

Implement new changes
`git merge`
